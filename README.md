# 📊 Dashboard de Sistemas Operativos

Este proyecto es un **dashboard interactivo** desarrollado con **Angular** y **ng-apexcharts**, que muestra la evolución del uso de diferentes sistemas operativos (Windows, Linux y macOS) desde 2001 hasta 2024.

---

## 🚀 Tecnologías utilizadas
- **Angular 17+**
- **TypeScript**
- **ng-apexcharts** (para los gráficos)
- **HTML5 / CSS3 (SCSS opcional)**
- **Interfaz responsive y moderna**

---



# Clonar y instalar
git clone [tu-repo]
////

////
npm install

# Ejecutar
ng serve
